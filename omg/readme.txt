1. Виконати адаптивну верстку.
1.1. Три блоки мають ставати в один рядок.
1.2. При ресайзі третій блок має перелаштовуватись на наступний рядок і займати ширину, рівну ширині першого рядку.
1.3. В фінальній стадії ресайзу всі блоки мають стати в один стовпчик.
1.4. Фонове зображення має бути зафіксованим і займати увесь простір.


2. Зробити кнопку "показати ще" (дизайн на власний розсуд).
2.1. При кліку на кнопку мають додаватись нові три блоки (за допомогою AJAX запиту до файлу items.json).
2.2. Шляхи у файлі можна змінювати.

3. Додати можливість вмикати/вимикати режим "нескінченного скролу" (дизайн на власний розсуд):
3.1. При вмиканні режиму, дозавантаження нових блоків відбувається тоді, коли юзер доскролює сторінку вниз. Кнопка "показати ще" зникає.
3.2. При вимиканні режиму, знов з’являється кнопка "показати ще". Додавання нових блоків відбувається по натисканню на цю кнопку.

4. Додати кнопку автоматичної прокрутки вверх (дизайн на власний розсуд).

Вимоги:
1. Кроссбраузерність (IE10, Chrome, Firefox, Opera).
2. При кліку на кнопку автоматичної прокрутки вверх - плавна(можно швидка, але все одно анімована) прокрутка.
3. Невикористання Bootstrap.

Буде плюсом:
1. Використання чистого js, без використання бібліотек.
2. Коментування ключових моментів, що відбуваються у javascript.
